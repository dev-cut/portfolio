# Intro 배지 플로팅 애니메이션 적용

## 작업 개요

사용자의 요청에 따라 Intro 섹션의 프로필 이미지 주변 배지(날짜, 국적)에 둥둥 떠다니는 듯한 플로팅 애니메이션을 적용했습니다.

## 작업 상세

- [x] `components/Intro.module.scss`:
  - `@keyframes float` 정의: 위아래로 부드럽게 움직이는 애니메이션
  - `.dateBadge`: 3초 주기의 플로팅 애니메이션 적용
  - `.nationalityBadge`: 3.5초 주기, 0.5초 지연의 플로팅 애니메이션 적용 (자연스러운 움직임 유도)

## 결과

배지들이 정지해 있지 않고 부드럽게 움직이며 생동감 있는 인터랙션을 제공합니다.
