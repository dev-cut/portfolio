# Experience 섹션 별 아이콘 플로팅 애니메이션 적용

## 작업 개요

사용자의 요청에 따라 Experience 섹션의 리스트 아이템에 있는 별 아이콘(✦)에 둥둥 떠다니는 플로팅 애니메이션을 적용했습니다.

## 작업 상세

- [x] `components/ResumeGrid.module.scss`:
  - `@keyframes float-marker` 정의: 위아래로 5px 정도 움직이는 부드러운 애니메이션
  - `.marker`: 3초 주기의 플로팅 애니메이션 적용

## 결과

Experience 섹션의 별 아이콘들이 부드럽게 움직이며 시각적인 흥미를 더해줍니다.
